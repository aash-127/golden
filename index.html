اهلا بك

اهلا بك في موقع شبكة الإبداع. هنا حيث يجتمع الإبداع والمهارات في هذا المكان المميز. وكذلك يمكنك مشاهدة فريق المطورين والمصممين من خلال الضغط على "الفريق".

اكتشف الكورسات الأكثر شيوعًا

HTML

هي اللغة الأساسية المستخدمة لإنشاء صفحات الويب. تُستخدم لتحديد بنية المحتوى على الصفحة مثل النصوص، الصور، الروابط، الأزرار، القوائم، الجداول وغيره

JavaScript

هي لغة برمجة ديناميكية تُستخدم أساسًا لتطوير الويب. تضيف التفاعلية لصفحات الويب، مثل النماذج المتحركة، معالجة الأحداث، والتحقق من البيانات.

Figma

هي أداة تصميم واجهات مستخدم (UI) وتعاون تعمل عبر الإنترنت. تُستخدم لإنشاء تصميمات رقمية، مثل تطبيقات الويب والهاتف

css

هي لغة تنسيق تُستخدم لتصميم مظهر صفحات الويب المكتوبة بـ HTML. تحدد CSS ألوان النصوص، الخطوط، الهوامش، الأحجام، الخلفيات، والتخطيطات.
